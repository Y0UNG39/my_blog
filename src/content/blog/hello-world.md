欢迎来到我的个人博客！这是我使用 Astro 搭建的第一篇文章。

## 为什么写博客？

写博客是一种很好的学习方式。把知识整理成文字，不仅能加深理解，还能帮助到其他人。

## 技术栈

这个博客使用了以下技术：

- **Astro** — 静态站点生成框架
- **Tailwind CSS** — 原子化 CSS 框架
- **Markdown** — 内容格式
- **Pagefind** — 客户端全文搜索

## 代码示例

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```
